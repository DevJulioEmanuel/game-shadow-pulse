# Shadow Pulse

**Engine:** GameMaker  
**Gênero:** Plataforma 2D + Puzzle + Boss Fight  
**Duração:** 5–7 minutos  

---

## Visão Geral

O jogador deve:

- atravessar uma fase de parkour
- resolver um puzzle ambiental
- enfrentar um boss final em duas fases

O projeto busca entregar uma experiência curta, intensa e memorável.

---

## Objetivo Técnico
Este projeto foi criado para demonstrar competências em:

- programação gameplay
- física 2D
- colisão
- máquinas de estado (state machine)
- IA de boss
- animações
- suporte a teclado e controle

---

## Fluxo do Jogo
Spawn  
→ Parkour  
→ Puzzle  
→ Boss Fase 1  
→ Transformação  
→ Boss Fase 2  
→ Tela de Vitória
